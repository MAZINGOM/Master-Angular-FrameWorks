# What is a Component based Architecture in Angular
1. A component based architecture involves defining tha components which are the building blocks of  any Angular application.
2. Components communicate with other components & generally , a component contains an HTML ,TypeScript & CSS file where we write the logic for the     page want  to display .

# What files does any basic Angular Component have ?
1. When ever a new component is cretaed apart from the root component it will have an HTML file , a CSS file , an TypeScript and a testing file .
